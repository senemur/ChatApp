# ChatApp - Real-time Chat Application

ASP.NET Core MVC, SignalR ve SQL Server kullanarak geliştirilmiş gerçek zamanlı sohbet uygulaması.

## Özellikler

-  Gerçek zamanlı mesajlaşma (SignalR)
-  Bire-bir ve grup sohbetleri
-  Kullanıcı kimlik doğrulama
-  Online/Offline durumu
-  Yazıyor göstergesi
-  Mesaj geçmişi

##  Teknolojiler

- ASP.NET Core 8.0 MVC
- SignalR
- Entity Framework Core
- SQL Server
- ASP.NET Identity
- Razor Views
- Bootstrap

## Mimari

Clean Architecture prensiplerine uygun katmanlı mimari:
- **ChatApp.Domain** - Entity'ler ve domain modelleri
- **ChatApp.Application** - Business logic ve servisler
- **ChatApp.Infrastructure** - Database ve data access
- **ChatApp.Web** - MVC Controllers, Views ve SignalR Hubs
